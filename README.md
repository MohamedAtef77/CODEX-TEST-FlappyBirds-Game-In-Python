# Flappy Bird (Python Edition)

This repository contains a desktop clone of the 2013 viral mobile game
**Flappy Bird**, implemented with [pygame](https://www.pygame.org/). Guide the
little bird through an endless series of pipes by flapping to stay aloft.

## Prerequisites

* Python 3.9 or later
* `pygame` (install with `pip install pygame`)

## How to Play

```bash
python game.py
```

* Press **Space** or the **Up Arrow** to flap.
* Avoid colliding with pipes or the ground.
* After a crash, press **Space** to restart.

Enjoy, and see how high you can score!
